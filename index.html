* Difference between div and span tag:

1. The div tag is a block level element while span tag is a inline element.
2. div tag accepts align attribute while span tag does not accepts align attribute.

* Difference between class and ID's:

A Class name can be used by multiple HTML element, while an ID name must only be used by one HTML element within the page.

* What are self closing tags?

A self closing tags in HTML is a kind of HTML tag that does need to be close manually by its closing tags which means it does not have seperate closing tags.

* How many types of heading are there? How to decide which to use where.

There are 6 types of heading. h1 headings should be used for main heading, followed by h2 headings then the less important h3.

* Difference between block and inline

Block elements always start from a new line. Inline elements never start from a new line. Block element cover space from left to roght as far as it can go. Inline element only cover the spaces as bound by the tags int the HTML elements

* Difference between em and strong

em tag is used for emphasizing the text as italic while strong tag is used for making the text as bolder.
